## Introduction
This project tests out automating the compilation of a C++ project using the [make](https://www.gnu.org/software/make/manual/make.html) and [g++](https://gcc.gnu.org/onlinedocs/gcc-13.2.0/gcc/). The below earthly tutorial is used to learn and utilise make

https://earthly.dev/blog/g++-makefile/

## Compiling and Running the project
if you have make and g++ installed simply run ```make``` or ``` make run``` in the root directory/folder and the program will compile and run.
